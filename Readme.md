## Oma arvio työstä ja oman osaamisen kehittymisestä
Mielestämme onnistuimme hakutoiminnallisuuden ja overlayn toteutuksessa hyvin.
Parantamista olisi responsiivisuuden hienosäätö ja hakuhistorian lisääminen.

## Tietoja sovelluksesta
**Elokuvahaku** on sovellus, joka hakee elokuvien tietoja OMDB API:sta.  
Käyttäjä voi kirjoittaa elokuvan nimen, nähdä hakutulokset korttimuodossa ja klikata korttia saadakseen lisätietoja overlayn kautta.
